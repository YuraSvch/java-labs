## Опис проєкту
Ця лабораторна робота присвячена вивченню умовних операторів (`if-else`, `switch-case`), обробці введення даних з клавіатури за допомогою класу `Scanner` та використанню вбудованих математичних методів класу `Math`. Програма також демонструє базову обробку винятків для уникнення помилок під час введення некоректних даних.

## Реалізовані завдання
Програма складається з одного класу `Lab2`, який містить кілька послідовних блоків коду у методі `main`:

1. **Завдання 1 (Квадратне рівняння):** * Реалізовано обчислення коренів квадратного рівняння за заданими коефіцієнтами `a`, `b` та `c`.
   * Додано перевірку на нульовий коефіцієнт `a` та аналіз дискримінанта (два корені, один корінь або їх відсутність).
   * **Особисте завдання:** Впроваджено цикл блок `try-catch` (`InputMismatchException`) для захисту від введення тексту замість чисел.

2. **Завдання 2 (Координатні чверті):**
   * Програма приймає координати точки `(x, y)` та за допомогою каскаду умов `if-else` визначає, у якому з чотирьох квадрантів вона знаходиться (або чи лежить на осях).

3. **Завдання 3 (Аналіз числа):**
   * Користувач вводить ціле число, після чого програма перевіряє дві умови одночасно: чи є число двозначним (використовується `Math.abs` для врахування від'ємних чисел) та чи є воно парним (`% 2 == 0`).

4. **Завдання 4 (Математична функція та пори року):**
   * **4.1:** Обчислення кусково-заданої функції `f(x)` залежно від введеного значення `x`. Використано методи `Math.exp()`, `Math.pow()`, `Math.cos()`, `Math.sin()` та `Math.abs()`.
   * **4.2:** Визначення пори року (Зима, Весна, Літо, Осінь) за введеним номером місяця з використанням конструкції `switch-case`.
