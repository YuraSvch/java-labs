## Опис проєкту

Ця лабораторна робота присвячена вивченню принципів роботи з одновимірними масивами в мові Java. Програма демонструє створення масивів, їх ініціалізацію, різні способи перебору елементів за допомогою циклічних конструкцій (`for`, `while`), а також виконання базових математичних операцій над елементами масиву (знаходження суми, зміна значень за умовою). 

**Вимоги до середовища:** Для компіляції та успішного запуску коду необхідний встановлений **JDK (Java Development Kit)**.

**Загальна реалізація:** Код структуровано за принципами об'єктно-орієнтованого програмування. Проєкт складається з головного класу `Lab4_Main`, який слугує точкою входу, та чотирьох окремих класів (`Lab4_Task1` – `Lab4_Task4`) для кожного із завдань. Головний клас послідовно створює екземпляри класів-завдань та викликає їхній метод `execute()`, що робить код модульним та зручним для читання.

## Реалізовані завдання

1. **Завдання 1 (Створення та ініціалізація):** * Створено два цілочисельних масиви розміром по 50 елементів.
   * Перший масив заповнюється першими 50-ма парними числами, другий — непарними. Використано цикл `for` для автоматизації розрахунку та запису значень.

2. **Завдання 2 (Методи перебору масиву):** * Робота із заданим масивом цілих чисел.
   * Продемонстровано 5 різних підходів до ітерації: 
     - Повний перебір циклом `while`.
     - Повний перебір циклом `for`.
     - Виведення елементів лише з непарними індексами (через `while`).
     - Виведення елементів лише з парними індексами (через `for`).
     - Виведення всіх елементів масиву у зворотному порядку.

3. **Завдання 3 (Сума елементів):** * Робота з масивом дійсних чисел типу `double`. 
   * Реалізовано алгоритм підрахунку загальної суми всіх елементів масиву за допомогою циклу та змінної-акумулятора.

4. **Завдання 4 (Модифікація масиву):** * Реалізовано обробку масиву за заданою умовою: програма знаходить усі непарні числа в масиві та змінює їхній знак на протилежний. 
   * Додано допоміжний метод `printArray` для зручного виведення стану масиву до та після модифікації.

## Як запустити локально

1. Відкрийте термінал та перейдіть у кореневу папку проєкту (де знаходиться папка `lab4`).
2. Переконайтеся, що у вас встановлено JDK (можна перевірити командою `javac -version`).
3. Скомпілюйте одразу всі класи лабораторної роботи:
   
   javac lab4/*.java

4. Запустіть головний файл програми:

   java lab4.Lab4_Main
